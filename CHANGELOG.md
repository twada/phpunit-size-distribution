## [0.1.0] - 2026-01-18

### Added

- Test size distribution reporting for PHPUnit test suites
- Support for `#[Small]`, `#[Medium]`, and `#[Large]` test size attributes
- Console output showing counts and percentages for each size category
- Counting of Passed, Failed, and Errored tests (Skipped tests are excluded)
- Support for PHPUnit 10.5+, 11.x, and 12.x
- Support for PHP 8.1+
